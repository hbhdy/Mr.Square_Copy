---------
게임 소개
---------
+ 장르 : 퍼즐
+ 이 게임은 2차원을 바탕으로 바닥의 타일을 모두 색칠하는 게임입니다.

---------
실행 방법
---------
+ 압축 해체 후 Release폴더 안에 있는 MrSquare.exe 파일을 실행 하면 됩니다.
+ 조작법은 마우스 클릭으로 메뉴를 선택하고 키보드 방향키로 게임을 플레이 합니다.
+ 치트키는 키보드 숫자 0,1,2,3,4를 눌러 캐릭터를 바꿀 수 있게 했습니다.
